# Spotify Manager

Web app to manage your Spotify library by adding labels (tags).

# Getting started local development

## Install dependencies

```sh
$ npm install
```

## Run database

```sh
$ docker-compose up -d
```

## Prisma

```sh
# studio
$ npx prisma studio

# migrate
$ npx prisma migrate dev
```

## Run app

```sh
$ npm run dev
```
